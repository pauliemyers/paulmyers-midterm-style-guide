Paul Myers' Portfolio Style Guide

This style guide is meant to inform the creation of web pages as this portfolio grows.

Text

This portfolio is meant to be read almost like a menu or a pamphlet. As such, two fonts are proposed. All major headings should be written in "Roberto Condensed", sans-serif. All other headings should be written in "Open Sans", sans-serif.

Navigation via Images

This web page requires many image links to work. Every page that is not a "content" driven page should have thumbnail images that serve as previews of the page to which clicking on it will lead. As such, there are some rules to follow while creating images.
  • Home Page

Colors

This website relies upon colors to be interesting. Each different area on the website uses the same color patterns, but the colors themselves differ. Some general rules:
  • The page will always load in its lightest shade. Text will always be in the darkest shade.
  • On "Content" pages, the page will always be in the a very light gray to not influence the perception of the content. The text will be in the "dark" shade of the given color. Descriptive text should always be in a box of the "dark" shade, located to the right of the image. "dark" shaded boxes will always have a the light gray font color.
  • The footer of the page will always be a box of the "darker" shade. This "darker" shade is the darkest of the three provided shades, and its text will always be the lightest shade of the given color.

Logo

Logo Placement

The logo of the page is always displayed in the top-left corner of the website, and it should always be sized with a width of 20% and a height of 10%. Its position should be relatively positioned.

Logo Color

Logos should always be monochrome in the "dark" color of a given page except for main and streaming pages. In that case, the multi-colored logo should be used.
